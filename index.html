Crowd Counting

CNN / Tensorflow / Transfer Learning

A Tensorflow implementation undertaken for crowd counting using overhead surveillance cameras. Useful in flow analysis for optimizing product placement. Also applicable for use to prevent crowding in public areas during the relaxation of lockdowns. Model Optimization

TensorRT / PyTorch / NVIDIA

TensorRT is a C++ framework that optimizes the network by combining layers and optimizing kernel selection to facilitate inference on NVIDIA GPUs in an optimal fashion. Trained the YOLOv3 network using the PyTorch framework on the CIFAR-10 dataset. The trained model was converted to an ONNX model, then into a CUDA engine and TensorRT's C++ API was used to run inference. Resulted in ~4x reduction in latency on the test set with no change in the accuracy.
